# 쇽(SSUK)
쇽은 <mark>사용자의 반응 시간을 측정하고, 이를 연령대별 기준과 비교하여 결과를 제공하는 반응 속도 테스트</mark>입니다. 사용자는 화면의 색이 변하는 순간에 반응하여 클릭하며, 자신의 반응 속도를 실시간으로 확인할 수 있습니다.
